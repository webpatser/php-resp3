# Code of conduct

This project follows the [Contributor Covenant 2.1][covenant]. The full text
lives at the link below; this file summarises what it means in practice.

## In short

Be kind, assume good faith, give clear feedback. Disagree on technical
points without making it personal. Help newcomers when you can. The
maintainers do the same back.

## Reporting

Send a private email to `oss@downsized.nl` if you see behaviour that
breaks the covenant. Reports stay confidential by default. The maintainer
acknowledges within seven days, investigates, and follows up with the
reporter on the outcome.

If the report concerns the maintainer, send it to a person you trust in
the broader PHP community and ask them to escalate; you do not have to
work with the subject of the report alone.

## Scope

The covenant applies to project spaces (this repository, its issues,
pull requests, discussions, related chat) and to public spaces when an
individual is representing the project (a conference talk, a blog post
that names the project, etc.).

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
